# Heart Icon

A simple HTML and SVG project built as part of my coding practice.

## About the Project

This project is a basic webpage that displays a red heart icon using SVG.

It was created to practise embedding scalable vector graphics directly into HTML and to understand how simple shapes and paths can be used to create icons.

## Features

- Displays a heart icon using SVG
- Uses an inline `svg` element
- Uses a `path` element to define the heart shape
- Simple and lightweight HTML structure

## Built With

- HTML5
- SVG

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Embedding SVG directly into HTML
- Using the `svg` element
- Understanding the `viewBox` attribute
- Using the `path` element to draw custom shapes
- Styling SVG with fill color
